We are the Player Piano software engineering team.


Project_Synopsis: The Player Piano project plans to create a highly accessible piano playing application that can connect various USB based accessible control devices to a real piano. In addition, we will develop a desktop application that displays a highly intuitive user interface that focuses on easy accessibility for users that have difficulty with traditional input methods.
